# Finance Manager (PySide6)

**Finance Manager** is a desktop financial management application built
with **PySide6**.  
It allows users to track personal **income and expenses**, analyze
financial activity over selected periods, and view visual statistics.

The application uses **PostgreSQL** as the main database for reliable and
persistent data storage.
